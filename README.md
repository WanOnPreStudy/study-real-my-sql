# study-real-my-sql
### 도서 "Real MySQL" 스터디

### 공부거리 : [Real MySQL 8.0](http://www.yes24.com/Product/Goods/103415627)
### 진행 방식
1. 스터디 장이 매주 공부할 분량을 정해 이슈 발행한다.
2. 정해진 분량을 학습 후 정리한 내용이나 실습 코드 PR 요청으로 인증한다.
### 발표
1. 매주 1명이 발표한다.
2. 발표 후, 토론한다.
---

### PR 메세지 규칙
1. 제목 규칙 : 이름_차수  
  ex) 홍길동_1주차
2. 내용 규칙 : 
  - 이슈 번호를 반드시 기입한다
